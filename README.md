# EmpSystem
A Java Web App developed while learning the basics of Spring Framework, Bootstrap Framework & Thymeleaf Template Engine.

## Introduction
- EmpSystem is an Employee Management System with Basic CRUD Capabilites.
- It features an index of employees with the option to add, edit & remove employees.

## Project Details

### Pages
This web app consists of 3 pages viz. Employee List, Add Employee & Edit Employee.

#### Employee List
- The Employee List page serves as an index of all the employees in the database.
- It features options to edit or delete employees from the database.
- The Edit button redirects to the Edit Employee page with the Employee details.
- While the Delete button reloads the page after database update.
![Employee List Page](https://github.com/user-attachments/assets/fe6b7af5-0e7d-4d26-8c4d-31d069ea3348)

#### Add Employee & Edit Employee
- The Add Employees page is used to add an employee to the database. On Submit, it redirects to Employee List.
- Along with being used for adding new employees to the database, the same layout is also used to edit employees.
- For editing, the employee details are fetched by id. And, it redirects back to Employee List page after database update.
